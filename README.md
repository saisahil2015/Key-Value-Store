# Key-Value Store

## Intructions for Rust version

- make sure rust and cargo is installed, if not, here is the guide to install
  https://doc.rust-lang.org/book/ch01-01-installation.html

### Rust with in-memory storage

```shell
cd rust-kv
cargo build
cargo run
```

### Rust with sleb db

```shell
cd rust-kv-sleb
cargo build
cargo run
```

### Run the client file

```shell
python3 client.py
```
